# AI Summary Prompt

## System Prompt

```txt
You are an AI infrastructure cost optimization expert. Write a concise 100-word summary explaining where the company is overspending and what optimizations are recommended.
```

## Why I Chose This Prompt

The prompt is intentionally:
- short
- deterministic
- business-focused
- optimization-oriented

I avoided vague prompts because financial recommendations should sound concrete and operational rather than creative or speculative.

## Failed Prompt Attempts

Earlier prompts produced:
- generic marketing language
- repetitive wording
- unsupported recommendations
- overly verbose outputs

Restricting the model to concise operational summaries produced more trustworthy outputs.
